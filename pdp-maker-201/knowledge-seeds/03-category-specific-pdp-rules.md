제품 사실 대체 금지: 이 문서는 카테고리별 체크리스트만 제공한다. 실제 성분, 소재, 규격, 인증, 후기, 기능, 사용 조건은 제품 자료가 있을 때만 확정한다.

목적: 제품 카테고리별로 상세페이지가 반드시 다뤄야 하는 구매 기준, 신뢰 근거, 주의 표현을 정리한다.

## 뷰티 / 화장품

구매자는 피부 타입, 사용감, 향, 발림성, 흡수감, 성분, 자극 가능성, 사용 순서를 본다. 이미지에서는 제형, 텍스처, 사용 장면, 용기 디테일을 보여주는 것이 좋다.

피부 개선, 미백, 주름, 여드름, 아토피, 홍조, 재생, 치료, 항염, 살균, 독소 제거, 체중감량 등 의학적 효능처럼 보이는 표현은 근거와 허용 범위가 없으면 쓰지 않는다.

권장 섹션: 제형/사용감, 피부 타입별 추천, 사용 순서, 전성분 또는 핵심 성분 설명, 패치 테스트 안내, 용량/사용 기간, 주의사항.

## 식품 / 건강식품

구매자는 맛, 원재료, 원산지, 영양 정보, 섭취 방법, 보관 방법, 유통기한, 알레르기 정보를 본다. 건강 관련 제품은 기능성 인정 여부와 표현 범위가 중요하다.

질병 예방, 치료, 체중 감량 보장, 혈당/혈압/콜레스테롤 개선 보장 같은 표현은 피한다. 건강기능식품이 아닌 제품을 건강기능식품처럼 보이게 하면 안 된다.

권장 섹션: 원재료와 맛, 섭취 장면, 1회 섭취량, 보관법, 알레르기 주의, 영양성분, 포장 단위, 선물/대량 구매 활용.

## 패션 / 잡화

구매자는 핏, 사이즈, 소재감, 두께, 비침, 신축성, 무게, 계절감, 세탁법, 컬러 차이를 본다. 제품 이미지는 착용 컷, 디테일 컷, 소재 질감, 사이즈 비교가 중요하다.

모델 착용 이미지는 실제 상품과 다르게 보일 수 있으므로 사이즈표와 실측 정보를 함께 제공한다. “누구나 완벽한 핏” 같은 표현은 피하고 체형별 선택 기준을 제공한다.

권장 섹션: 착용 실루엣, 소재 디테일, 사이즈 선택법, 컬러 옵션, 세탁/관리, 코디 제안, 배송/교환 주의.

## 생활용품 / 주방 / 홈

구매자는 크기, 설치 가능 공간, 소재, 내구성, 청소/관리, 사용 전후 정리감, 안전성, 호환성을 본다. 제품의 실제 스케일을 보여주는 이미지가 중요하다.

권장 섹션: 사용 전 문제, 설치/사용 장면, 사이즈 비교, 소재와 구조, 청소 방법, 보관 방법, 구성품, 사용 시 주의사항.

## 전자제품 / 디지털 액세서리

구매자는 호환 기기, 전원 방식, 배터리, 출력, 연결 방식, 크기, 무게, 보증, AS, 안전 인증을 본다. 스펙 오류가 반품과 CS로 이어지기 쉬우므로 확인된 정보만 쓴다.

권장 섹션: 호환성, 핵심 기능, 사용 단계, 스펙표, 구성품, 안전/인증, AS 안내, 자주 묻는 질문.

## 소프트웨어 / SaaS / 앱 / 웹서비스

구매자는 기능 수보다 "내 업무가 어떻게 달라지는지"를 본다. 대상 사용자, 해결하는 업무, 현재 방식의 불편, 사용 흐름, 데모 화면, 요금제, 무료 체험, 보안, 개인정보, 연동, 도입 지원, 해지/환불 조건이 핵심 구매 기준이다.

스크린샷이나 기존 서비스 화면이 있으면 UI 구조, 메뉴명, 화면 색상, 실제 기능명을 보존한다. 제공되지 않은 고객사 로고, 보안 인증, 성과 수치, 요금표, 연동 목록, 가짜 대시보드 데이터는 만들지 않는다.

권장 섹션: Hero 가치 제안, 업무 문제 공감, 3단계 데모, 핵심 기능과 이득, 역할별 사용 사례, 보안/신뢰, 요금제/무료 체험 CTA, 도입 전 FAQ.

## 유아 / 키즈

구매자는 안전성, 소재, 연령, 세척, 피부 접촉, 삼킴 위험, KC 등 인증 여부, 부모 사용 편의성을 본다. 아이가 쓰는 제품은 과장된 안전 표현보다 확인 가능한 정보가 중요하다.

권장 섹션: 권장 연령, 소재/마감, 세척법, 안전 주의, 사이즈, 사용 장면, 보호자 관리 포인트.

## 반려동물

구매자는 반려동물 크기, 품종, 알레르기, 내구성, 세척, 안전 소재, 급여량 또는 사용법을 본다. 건강 개선 표현은 수의학적 근거가 없으면 피한다.

권장 섹션: 대상 크기/종, 사용 또는 급여 방법, 소재/성분, 세척/보관, 주의사항, 교체 주기.

## B2B / 사무 / 산업용

구매자는 효율, 비용 절감, 규격, 납기, 대량 구매, 설치 조건, 유지관리, 인증, 견적 가능 여부를 본다. 감성보다 신뢰성과 명확한 스펙이 중요하다.

권장 섹션: 도입 문제, 핵심 기능, 규격표, 적용 현장, 설치/운영 조건, 견적/문의 CTA, 유지관리.

## 카테고리 공통 금지

확인되지 않은 “1위”, “최고”, “완벽”, “무조건”, “100%”, “영구”, “전문가 추천”, “인증 완료”, “임상 검증”, “후기 폭발”은 쓰지 않는다.

자료가 없는 경우 “제품 이미지에서 확인 가능한 범위”로만 설명하고, 불확실한 항목은 사용자 확인 필요로 남긴다.

## 검색 키워드

카테고리별 상세페이지, 뷰티 화장품, 식품 건강식품, 패션 잡화, 생활용품, 전자제품, 소프트웨어 상세페이지, SaaS, 앱 홍보, 웹서비스, 프로그램, 디지털 제품, 데모, 무료 체험, 요금제, 보안, 개인정보, API 연동, 팀 협업, 키즈 유아, 반려동물, B2B, 구매 기준, 필수 정보, 주의사항, 효능 표현, 스펙, 구성품, 안전 표현.
